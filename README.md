# 🎨 REAL TIME HAND WRITE ON CAMERA 
AI-Powered Hand Tracking Drawing App

# 🚀 Overview

This is a futuristic drawing application that lets you create digital art using only your hand gestures. Powered by MediaPipe and OpenCV, it transforms your webcam into a touchless canvas.

No mouse. No touchscreen. Just your hand. ✨

# 🧠 Features

✍️ Draw with your index finger

✌️ Erase with two fingers

🤌 Smooth zoom in/out with pinch gesture

✋ Move (pan) the canvas freely

🖐️ Save your artwork with pinky finger

🎨 Dynamic color palette & quick color selection

➕➖ Adjustable brush size

🧼 Clear canvas instantly

⚡ Real-time performance

# 🎮 Controls (Hand Gestures)
Gesture	Action
☝️ Index Finger	Draw / UI Interaction

✌️ Index + Middle	Eraser

🤌 Thumb + Index	Zoom

✋ Open Hand	Move Canvas

🖐️ Pinky Finger Only	Save as PNG

# 🛠️ Technologies Used

OpenCV – Image processing & rendering

MediaPipe – Hand tracking & gesture detection

NumPy – Fast numerical operations

Python – Core programming language

# 📦 Installation

pip install opencv-python mediapipe numpy

Download the required model file:
hand_landmarker.task

Place it in the project directory.

▶️ Run the Project
python main.py

# 💡 How It Works

Detects hand landmarks in real-time

Interprets finger positions as gestures

Maps gestures to drawing actions

Applies transformations like zoom & pan

Renders everything smoothly on screen
